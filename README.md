## ⚙ About the project

This Project is an online chat application to socialize online with others (Friends, family.. etc.)<br>
It is my passion project so I get back to it time to time and make something with it<br>
It's a relatively simple Django webapplication that uses the websocket feature of the Django framework to handle the communication between users.<br>
The server use the dockerized version of the project and looks like this in docker desktop:<br>
<img src="https://user-images.githubusercontent.com/90270578/234247049-c25fe1bf-919c-4af8-974b-48f15c51254c.png" width="500" />


## 🔧 The project use the following technologies:
  - Django
  - Jinja
  - JS
  - HTML
  - CSS
  - Docker

## 📋 Current feature plans and implementation status:

Implemented:<br/>

  ✅ Login/Register feature<br/>
  ✅ Relevant NAV bar<br/>
  ✅ Chat rooms to talk with others<br/>
  ✅ Showing current users in the room LIVE<br/>
  ✅ Responsive CSS design for PC and mobile devices<br/>
  ✅ Currently typing message, when somebody typing<br/>
  ✅ Emoticon menu for messaging<br/>
  ✅ User profile page<br/>
  ✅ Cool looking CSS design 😎
  
Not implemented <b> YET </b>(AKA: Future plans):<br/>

  ❌ Admin menu for moderation<br/>
  ❌ Delete message button for Admin role in chat rooms<br/>
